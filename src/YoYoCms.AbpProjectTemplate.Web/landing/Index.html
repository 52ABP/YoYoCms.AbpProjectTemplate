基于DDD的.NET项目生成器-ABP

ABP("ASP.NET Boilerplate Project ")是一个使用最流行技术开发的现代WEB应用程序，也是一个强大的基础设施框架。

功能概览

ASP.NET Boilerplate是一个最佳实践和使用最流行技术开发的现代WEB应用程序，它旨在成为一个通用的WEB应用程序开发框架，也是一个基于DDD(领域驱动设计)的经典分层架构，实现了众多DDD的概念。

多层架构体系

ABP为了减少系统的复杂性和提高代码的可重用性，采用分层架构 ，它也是一种被广泛接受的技术。

模块化设计

ABP框架提供了创建和组装模块的基础，一个模块能够依赖于另一个模块。在通常情况下，一个程序集就可以看成是一个模块。

领域驱动设计

ABP遵循DDD（领域驱动设计）的原则，将工程分为四个层：展现层(Presentation)、应用层(Application)、领域层(Domain)、基础设施层(Infrastructure).

多租户(SaaS模式)

开发SaaS化的应用程序更加的快捷和容易。实现了从数据库到用户界面的多租户系统。

完善的文档

全面的文档。(中文版)和（英文版）和丰富的入门教程。电话薄教程、简书教程合集

跨平台

ABP最新版本基于.Net Core，集成了ASP.NET Core,Entity Framework Core成为跨平台的基础设置架构。

技术支持

ASP.NET ZERO是ABP的一个商业版本，它可以通过提供预制的工作页面以及基于ABP强大的基础架构来节省您的时间。将其作为您的基础应用程序，并直接开始开发，专注于您的业务代码！

免费 & 开源

在Github开源，使用MIT许可证，正在积极开发中并开放贡献，欢迎提交代码。

代码生成器

ABP Code Generator 是基于ABP框架制作的代码生成器，可以用于大家在日常开发过程中节约时间，把更多的精力放于业务逻辑的处理中。 使用说明 l 下载地址.
